¿Cómo usar los datos en tus Handlers?
Una vez que el Middleware (que definimos en el paso anterior) guarda los datos con c.Set,
puedes recuperarlos en cualquier función de tus Handlers así:


func (h *UsuarioHandler) Listar(c *gin.Context) {
    // Recuperar el ID del usuario que hace la petición
    userID, _ := c.Get("userID")
    userRol, _ := c.Get("userRol")

    // Ahora puedes usar userID para filtrar datos o userRol para denegar acceso
    if userRol != "admin" {
        c.JSON(http.StatusForbidden, gin.H{"error": "No tienes permisos"})
        return
    }
    
    // ... resto de tu lógica
}

# Para subir a produccion el binario de kubo-api, estos comandos se tienen que correr en la carpeta de la aplicacion donde esta el Dockerfile
## Crear la imagen de Docker en mi Mac
docker build --platform linux/amd64 -t kubo-api:v1 .

## Emaaquetamos la imagen
docker save -o kubo-api.tar kubo-api:v1

## Probamos la imagen localmente
docker run -p 8080:8080 kubo-api:v1

## Subimos la imagen a Alibaba Cloud
