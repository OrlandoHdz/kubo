
# Kubo Backend

Este repositorio contiene la lógica del servidor (backend) para **Kubo Web**, encargado de gestionar la API, la conexión a base de datos y la lógica de negocio central.

## 🚀 Comandos Disponibles

Para facilitar el desarrollo en **Alialloys - Kubo**, se han configurado los siguientes comandos:

| Comando | Descripción |
| :--- | :--- |
| `help` | Muestra la lista de todos los comandos disponibles en el proyecto. |
| `run` | Inicia la ejecución de la aplicación principal (API). |
| `seed` | Puebla la base de datos con datos de prueba iniciales (usuarios, clientes, etc.). |
| `test-db` | Verifica la conexión a **PostgreSQL** utilizando la configuración actual del entorno. |
| `sqlc` | Genera automáticamente el código en **Go** basado en las definiciones de los archivos SQL. |

---

## 🛠️ Tecnologías Principales

* **Lenguaje:** Go (Golang)
* **Base de Datos:** PostgreSQL
* **Generación de Código:** sqlc
* **Entorno:** Alialloys
